# ZongziTEK 黑板贴连接器

该插件能够让 ZongziTEK 黑板贴 与 ClassIsland 进行连接与交互。
</div>

## 插件功能

- 将 ClassIsland 当前显示的课程表发送到黑板贴，并在黑板贴显示
- 合理切换黑板贴的显示状态
